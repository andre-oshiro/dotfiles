# dotfiles

## How to use

- 前提条件
  - Homebrewインストール済み
  - Gitインストール済み

### Steps

- リポジトリをclone
  - `git clone git@github.com:andre-oshiro/dotfiles.git`
- `setup.sh`を実行
- `setup.sh`が正常終了したら、iTerm2を起動
  - iTerm2 のフォントを変更
    - Preferences ⇨ Profiles ⇨ Text
  - `com.googlecode.iterm2.plist` preferenceをロード
    - Preferences ⇨ General ⇨ Preferences
