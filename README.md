# dotfiles

## 使い方メモ

### ステップ１

- **git cloneして `setup.sh` を実行**

```shell
$ git clone git@github.com:andre-oshiro/dotfiles.git ${ directory }
$ cd ${ directory }/dotfiles
$ sh ./setup.sh
    .
    .
 ~ 省略 ~
    .
    .
完了
$
```

### ステップ２

- **iTerm2 のフォントを変更**
  - **iTerm2 → Preferences... → Profiles → Text → Change Font**
    - FontFamily: `Noto Moto for Powerline`
